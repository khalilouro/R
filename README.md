# R

Ce dossier contient tous les **scripts R** utilisés dans le projet.

Les fichiers présents dans ce dossier implémentent la logique principale de l'application, notamment :

* les fonctions de conversion de bases (`DecodeNumber`, `EncodeNumber`)
* l'encodage en système **Bibi-binaire**
* les traitements nécessaires au fonctionnement de l'application Shiny

Ces scripts sont écrits en **R scripting** et sont utilisés par l'application principale.
