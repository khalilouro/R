# Estimation de la moyenne avec R Shiny

## Description

Cette application Shiny permet d'estimer la moyenne d'une population à partir d'un échantillon.

L'utilisateur peut entrer ses propres données et obtenir :

- la moyenne de l'échantillon
- un intervalle de confiance à 95 %
- un histogramme des données

## Technologies utilisées

- R
- Shiny

## Lancer l'application

Installer Shiny :

install.packages("shiny")

Puis lancer l'application :

runApp("app.R")
